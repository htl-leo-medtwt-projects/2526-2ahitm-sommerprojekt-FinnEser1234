
# Projekt-Ordner

Integriere in diesen Bereich deine gesamte praktische Projektstruktur!
Jeglicher Fremdcode (z. B. Codepen, ChatGPT und CO.) ist als solcher im Projekt zu kennzeichnen.
Ergänze sprechende Kommentare in deinem Code zur Strukturierung + Erläuterung!

## Aktueller Stand

- Das Spiel enthält nun Punkte, Timer, Lives und einen Abschlussdialog mit Verdächtigen-Auswahl.
- In den Einstellungen werden gespeicherte Spielstände aus `localStorage` nach benötigter Zeit sortiert angezeigt.
- Der Übergang zum nächsten Raum erscheint erst, wenn alle klickbaren Elemente im Raum gefunden wurden.
